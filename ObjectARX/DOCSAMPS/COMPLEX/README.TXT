
Refer to ARX Developer's Guide

To use complex.arx:

There are four commands defined by complex.arx:

MAKEABLOCK
CREATEPOLY
ADDBLKWITHATT
PRINTALL

1. Start AutoCAD and open a new drawing.

2. Type the ARX command and load complex.arx.

3. Execute the commands in the order given above. For the ADDBLKWITHATT command 
give an insertion point of zero, a rotation angle of 45 and a text height of .2.
