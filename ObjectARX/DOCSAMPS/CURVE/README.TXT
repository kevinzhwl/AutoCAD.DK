
Section: Curve Functions
Functions projectEllipse and offsetEllipse

1.      Create an ellipse in AutoCAD.
2.      Load the arx application.
3.      Command: curves (This creates a new ellipse depending on which option
	is selected; Project creates an ellipse which is the result of a
	projection onto the plane with normal <1,1,1>, and Offset creates one
	which is offset 0.5 drawing units from the original.)
