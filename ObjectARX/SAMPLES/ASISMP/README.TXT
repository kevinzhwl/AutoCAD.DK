This is a sample application using the AutoCAD SQL Interface API.  

  Commands			Description
  --------			-----------
  ASISETUP			Configures the sample program.
  ASICONNECT			Connects to an SQL Environment
  ASIDISCONNECT			Disconnects from an SQL Environment
  ASISQLED			Interactive SQL editor.
  ASISCRIPT			SQL script processor
  ASICREATEENV			Creates an SQL Environment
  ASIDROPENV			Drops an SQL Environment

  In order to execute an SQL statement, you have to connect to SQL environment
  first, using the ASICONNECT command, and then invoke ASISQLED command.

  When using the ASISQLED command, end SQL statements with a semicolon.
