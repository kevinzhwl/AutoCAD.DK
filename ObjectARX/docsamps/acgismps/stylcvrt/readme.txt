  getTextStyle() (AcDb->AcGi style)
  
  This program demostrates converting back and forth between
  AcDb and AcGi text styles.

Procedure:
 1. Load stylcvrt.arx
 2. Command: ASDKSTYLECVRT
 3. Command: ZOOM extent

   "Test string" should appear in the graphics screen
   that is slightly slanted upward.

   The Developer's Guide and the source code explains what took
   place in more detail.
