  worldDraw() (textstyle use)
  
  This program demonstrates the use of AcGi Text styles.

Procedure:
 1. Load textstyl.arx
 2. Command: ASDKSTYLESAMP

    "This is a percent, '%%%'." should appear twice in the graphics
    screen. Both are upside down, one is oblique with bounding box.

    The Developer's Guide and the source code explains what took
    place in more detail.
