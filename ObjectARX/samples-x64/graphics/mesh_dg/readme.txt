Description: 
 
worldDraw() (mesh generation)
  
This example demonstrates the use of the mesh() function.


Procedure:
1. Load mesh.arx.
2. Command: ASDKMAKEMESH

3 by 3 faces with different colors are created

The ObjectARX Developer's Guide and the source code explain what took place in more detail.
