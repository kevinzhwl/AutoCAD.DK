Readme.txt

This directory contains the ObjectARX samples. The directories below the samples folder 
correspond to categories that the samples demonstrate. Readme files in the sample directories explain what techniques and APIs are exploited by each sample. In addition, the root samples directory contains an HTML readme that provides a brief description of each sample. To view this overview document, open SamplesReadMe.htm in your browser. (Microsoft(R) Internet Explorer(TM) is recommended, but not required.)
