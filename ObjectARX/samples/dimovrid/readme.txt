Dimension Override sample

This program demonstrates how to add and or modify dimvar overrides attached
to dimensions, leaders, or feature control frames.

The sample currently registers the following command:

-DIM_OVRD: allows the user to choose a dimension, specify a dimension
           variable group code, and assign it a new value.
