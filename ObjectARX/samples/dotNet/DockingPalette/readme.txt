		DockingPalette - .NET API Sample

This sample demonstrates how to create a docking palette using C# as well as drag and drop support in the .NET API.
The following commands are defined:

PaletteDemo - Creates a palette and shows it on screen. Simple command line prompts are supported to manipulate some of the palette properites.

netdrop - Helper command that is executed when the user drags the text from the palette to the AutoCAD screen.

To run the sample.  Build the application, and load it into AutoCAD using the NETLOAD command.  You will find the dockablepalette.dll module in the \bin\{Debug|Release} subfolder.
