Summary description for "GiMeta" sample.
 To run this sample project create the references to acdbmgd, acmgd
 The steps to run this application :
 1. Draw an entity (circle, ploygon etc.) 
 2. Call "NetLoad" from the command prompt
 3. Load Filer "GiMeta.dll"
 4. Call "mygimeta" from the command prompt
 5. Select an entity
 Result : The application will dump the information at the command prompt
