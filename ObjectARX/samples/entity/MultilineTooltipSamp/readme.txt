MultilineTooltipSamp sample

This program demonstrates how to create multiline tooltip in AutoCAD.
It demonstrate the use of an AcEdInputPointMonitor reactore to achieve that goal.

The sample currently registers the following command:

-MTOOPTIPON:  Activate the multiline tooltip support.
-MTOOPTIPOFF: Turn off the multiline tooltip support.
