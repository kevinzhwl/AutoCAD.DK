Description:
 
worldDraw() (shell generation)
  
This example demonstrates the use of the shell() function.


Procedure:
1. Load shell.arx.
2. Command: ASDKMAKESHELL

Red and blue adjacent faces are created.

The ObjectARX Developer's Guide and the source code explain what took place in more detail.
