Description:
 
getTextStyle() (AcDb to AcGi style)
  
This program demonstrates converting back and forth between AcDb and AcGi text styles.

Procedure:
1. Load stylcvrt.arx.
2. Command: ASDKSTYLECVRT
3. Command: ZOOM extent

"Test string" that is slightly slanted upward should appear in the graphics screen.

The ObjectARX Developer's Guide and the source code explain what took place in more detail.
