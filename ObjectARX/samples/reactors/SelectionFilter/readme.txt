	Example showing how to use the Enhanced SelectionSet API to highlight a nested entity
	inside of an XRef.


1. Open a drawing with an XRef attached
2. Load SelectionFilter.arx
3. Command test
4. Select an entities that belong to the XRef
5. The selected nested entities will be highlighted
6. Hit enter to complete the selection. Each ObjectId of the selected entites
   is printed on the command line.
